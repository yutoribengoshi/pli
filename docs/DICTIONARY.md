## 概要

PLIに内蔵される法律用語辞書は、機械翻訳の誤訳を上書きする「最後の砦」として設計されています。
特に法律用語は一般の機械翻訳では誤訳されやすく（例：強盗⇄窃盗、故意⇄過失、保釈⇄保証）、
辞書側で訳語を固定することで、現場で起きうる誤訳事故を防ぎます。

## 設計思想

### 機械翻訳との「ハイブリッド」運用

PLIは機械翻訳エンジン（OPUS-MT / NLLB-200 / ローカルLLM）をまず実行し、
その結果に対してこの辞書で**ポストプロセス置換**をかけます。

```
入力: 「彼は強盗ではなく窃盗です」
  ↓ NLLB-200 翻訳
中間: "He is not a robbery but theft"  ← 文法は微妙だが用語は出る
  ↓ Qwen2.5-72B JA→Urdu
一次: یہ ڈکیتی نہیں ہے، بلکہ چوری ہے
  ↓ 辞書照合（強盗=ڈکیتی, 窃盗=چوری が一致）
確定: 訳語フィックス、弁護人画面に表示
```

### 中間英語の併記

ウルドゥー語など弁護人が読めない言語でも、英語段階を画面に併記することで、
"robbery" と "theft" のように**英語レベルで誤訳を即座に検知**できます。

### 未知語ハイライト

辞書にない固有名詞・俗語・方言は画面でハイライト表示し、
弁護人が確認するまで翻訳を確定させない設定が可能です。
